About this Site:

This site (which has the working title "art-pages") is designed to act half as an ongoing front-end web development portfolio project and half as a functional resource for a handful of studio artist's.


As a Web Development Project...

As a portfolio project, art-pages is designed to showcase basic proficiency in responsive mobile-first web design implementing Bootstrap 3.3.7.

In the coming months the project will hopefully continue to expand into a showcase for proficiency in Bootstrap 4 and basic back-end development by incorporating an in browser user interface allowing studio artists to customize their own webpages and upload content independantly.

As a Resource for Studio Artists...

As a resource for studio artists, art-pages is simply meant to streamline the process of making professional websites for artists.

In it's current state artists who have been individually invited begin by organizing their portfolio content into a uniform file directory (folder) structure using the instructions on the Get Started page linked above, then sending a compressed version of their organized content directly via email.


As a Donation-Based Service...

Although development of webpages for studio artists is usually provided by frelance developers at an hourly rate or by automated platforms like Square Space for a monthly rate, art-pages is a purely donation-based and has no manditory contribution for the original development or subsequent updates associated with an artist's website.

Artists are asked to pay whatever they can whenever they can to support the project, and as it gains momentum, the contributions made will work directly towards the time and class-fees associated with the development of features that will streamline the process of building and maintaing the websites of all current and future participating artists.

If the project stays financially viable the goal is to quickly develop an in-browser user interface (like that of Square Space, Other People's Pixels, Wix etc.) that will allow artists to re-design and add new content to their personal websites independantly, without the use of any direct knoledge of html/css.

Thank you for visiting and thank you for your support!
