Starting Your Website:

These are the basic instructions you guys will need to get your files organized for your website. Doing it this particular way just makes sure I know what order you want things in, what text you want where, and makes everything usable with the website templates I've put together.

Hopefully the pictures are enough to get you through most of this, but if anything's not making sense you can click on the text buttons for more info. And of course you can call me too.

Sorry in advance if anything's confusing (!) also don't be shy about letting me know if you hit any points of confusion or catch any overlooked spelling errors, there's bound to be at least a few...


Step 00:

If you haven't checked out the Site Templates section linked above take a minute to look through it and figure out which template you'd like me to use for your site. Keep in mind you can change the font used. You can also choose which linked pages you'd like to have and what text appears for them ("Information" "Links" etc.). The linked sections shown on the templates are just examples.



Step 01:

Make a new folder on your computer's desktop and title it with "Your Name" followed by "Website" (i.e. "Tom's Website" or "Alex's Website" etc). That one was easy!

Step 02:

Inside your new website folder, create a series of additional folders for each link you want to appear on your site, numbering them to indicate which one you'd like to appear first, second, third etc.

You should name them whatever you want, they'll appear exactly as you write them including capital letters. Some examples might include:

  1. Home / Homepage
  2. Images / Portfolio
  3. New Work / Old Work / Archive
  4. CV / Resume / Information
  5. Contact / Links

Step 03:

If you'd like to have a single image appear as the first thing that comes up on your website (sometimes called a homepage or landing page), make sure you have a folder labelled "1. Home," "1. Landing Page" etc. and add a single image to this folder. It can just be named "Image01.jpg".

Double-check to make sure your image is a JPEG between around 750 and 4500 pixels.

Step 04:

Since your desktop file structure is mimicking a website, you'll want to add your images to the appropriate folder you've set up. For example, add the correct images to "2. New Work," or "2. 2018 Paintings," or "6. Drawings," or however you've named your link folders.

The image file names will determine the exact order they appear in on a given page. (i.e. "Image1.jpg" will appear first, "Image2.jpg" will appear second etc).

Please take your time to make sure the order of these images is exactly the order you want them to appear in on your webpage! Remember the order they'll appear in on your webpage is determined by the "Image1.jpg" "Image2.jpg" (etc) titles you give them.

Again, check to make sure all of your images are JPEGs between 750 and 4500 pixels.

Step 05:

All the information that you would like to appear underneath the images of your work (i.e. Title, Size, Materials, Date) should be put in a seperate text or PDF file. You can name your document something simple like "All-Image-Info.txt" or "ImageText.pdf" (etc.).

In this document you'll make one long numbered list for every image that lives in that particular folder. The text will appear exactly how you write it underneath the image that corresponds to the number you wrote in the document. Iif you can help it though avoid using italics or bold.

Make a new text document for each folder that has images in it. If this text file is not included in a folder, the assumption will be that your work should have no accompanying information, which may be the case on pages like "Home" or "Home Page."

Step 06:

If you want to have a Resume or CV link, just make a numbered "CV" "Resume" or "Information" folder (the name is totally up to you).

In this folder add another text or PDF file with all the information you'd like to be displayed using the simplest formatting possible (avoid special fonts, indentations, columns, bullet-pointed lists etc. as they may not carry over to your page). Keep it as simple as possible!

The other option is to make the link open as a seperate PDF if you'd prefer that. The process is exactly the same for this (just put your PDF in a "CV" folder).

Step 07:

Take a minute to think about what other linked pages you want to appear on your website and make folders for them, repeating Steps 2-6 adding numbered images, text documents with image information etc. Make sure to add a new text file in every folder if you want image info!

Ideas for More Linked Pages:

  1. New Work / Old Work
  2. Portfolio 2010-2018 (or any years)
  3. Images by Project (rather than date)
  4. Biography / Bibliography
  5. Contact / Links

Step 08:

This step is important! Once you think you're done, go through the following checklist to make sure you're ready to send all of your info. The goal is to make sure it all looks good and you wonâ€™t require major updates right away.


Check List:

Check all your folder numbers! Remember they will determine the order of the links on your site.

Check all your image numbers in all folders! Remember they will determine their order on any particular page.

Check your image formats and sizes! All images should to be JPEGs between 750 and 4500 pixels.

Check your image information text files / PDFs! (See Step 05). Remember the order you put your image info. in is super important.

Step 09:

Click (or right-click) on the folder, and locate the "Compress" option either in the top menu or in the menu that appears when you second-click the folder with your mouse. This will be a little different on versions of Windows or Linux but should be similar.

After you select "Compress" a new file should appear near your folder with the same basic name, this is it's compressed version which hopefully will be small enough to email to me.

If you're a pro with DropBox you can ignore this step and just send me a link to the folder.

Step 10:

Now all you have to do is email your compressed folder. Please title the email with the same name as your folder (i.e. "Tom's Website" or "Alex's Website") and then attach the compressed version of your folder. It should be the one ending with ".zip".

Also in the email let me know what Website Template you'd like me to build your site with, a special font you found on Google and anything else special you'd like me to try to do.

If you're compressed file is too large to email me, the simplest thing to do is to set up a Drop Box account and send me a link to the compressed file / folder. Here's a link to Drop Box. That's it for now, when you're site's ready to go we'll talk about linking it to your official domain name ("www.tomchung.org" etc.).
